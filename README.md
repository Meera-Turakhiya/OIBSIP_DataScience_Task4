# OIBSIP_DataScience_Task4
Classifying emails as Spam or Ham using Natural Language Processing and Machine Learning | OIBSIP Task 4

### Objective
To classify emails as spam or ham using Natural Language Processing and Machine Learning.

### Dataset
The dataset consists of email messages labeled as spam or ham.

### Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP Techniques

### Steps Performed

1. Data Cleaning
2. Text Preprocessing
3. Tokenization
4. Feature Extraction
5. Model Training
6. Classification

### Results

The model accurately identified spam and non-spam emails.

### Conclusion

Spam detection systems improve communication security and reduce unwanted messages.
